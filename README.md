# Visudhan G - Portfolio

A modern, cinematic portfolio website showcasing backend development skills and projects.

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom styling with animations
- **JavaScript** - Vanilla JS with GSAP for animations
- **GSAP** - Professional-grade animations

## Features

- Animated loading screen
- Custom cursor with section-based color changes
- Smooth scroll animations
- Responsive design
- Dark sidebar navigation
- Interactive project cards
- Skill cards with hover effects
- Contact section with social links

## Sections

- **Hero** - Introduction with animated title
- **Work** - Featured projects
- **Skills** - Technical expertise
- **About** - Personal information & resume download
- **Contact** - Email & social links

## Deploy

Deployed on [Vercel](https://vercel.com)
